# 📖 Markdown转换工具 - 详细使用说明

## 🎯 工具简介

这是一个**纯前端Web工具**，可以将AI生成的Markdown内容快速转换为：
- **TXT格式**：去除所有Markdown语法，生成纯文本
- **Word格式**：保留格式（标题、粗体、列表等）
- **HTML格式**：转换为网页格式，便于预览

## 🚀 快速开始

### 方法一：直接使用（最简单）

1. **下载所有文件**到本地文件夹
2. **双击打开** `index.html` 文件
3. 浏览器会自动打开工具页面
4. 开始使用！

### 方法二：部署到服务器

1. 将所有文件上传到Web服务器
2. 通过浏览器访问 `index.html`
3. 可以分享链接给其他人使用

### 方法三：GitHub Pages（免费托管）

1. 创建GitHub账号（如果没有）
2. 创建新仓库，上传所有文件
3. 在仓库设置中启用GitHub Pages
4. 获得一个免费的网址，可以分享给任何人

## 📱 使用步骤

### 第一步：输入Markdown内容

有三种方式输入内容：

1. **粘贴按钮** 📋
   - 点击"粘贴"按钮
   - 自动从剪贴板读取内容
   - 适合从AI工具复制后直接使用

2. **上传文件** 📁
   - 点击"上传文件"按钮
   - 选择.md或.txt文件
   - 文件内容会自动加载到输入框

3. **手动输入** ✍️
   - 直接在左侧文本框中输入或粘贴Markdown内容

### 第二步：选择转换格式

根据需求选择：

- **📄 转TXT**：去除所有Markdown语法，生成纯文本
  - 适合：需要纯文本分享，不保留格式
  - 示例：`# 标题` → `标题`

- **📘 转Word**：保留格式，生成Word文档
  - 适合：需要保留标题、粗体、列表等格式
  - 生成：.docx或.doc格式（根据浏览器支持）

- **🌐 转HTML**：转换为网页格式
  - 适合：需要网页预览或分享
  - 生成：.html文件，可在浏览器中打开

### 第三步：下载文件

1. 转换完成后，右侧会显示结果
2. 点击"⬇️ 下载文件"按钮
3. 文件自动下载到本地
4. 可以分享给他人

## 💡 使用场景示例

### 场景1：AI对话内容分享

**问题**：从ChatGPT复制的内容带有很多Markdown语法（`**粗体**`、`# 标题`等），分享给朋友时看起来很乱。

**解决**：
1. 复制AI生成的内容
2. 打开工具，点击"粘贴"
3. 点击"转TXT"
4. 下载纯文本文件
5. 分享给朋友

### 场景2：制作正式文档

**问题**：需要将Markdown内容转换为正式的Word文档。

**解决**：
1. 输入或上传Markdown内容
2. 点击"转Word"
3. 下载Word文档
4. 在Word中进一步编辑

### 场景3：网页分享

**问题**：需要将Markdown内容发布为网页。

**解决**：
1. 输入Markdown内容
2. 点击"转HTML"
3. 下载HTML文件
4. 上传到网站或直接打开

## ❓ 常见问题

### Q1: 为什么Word转换失败？

**A**: 可能是浏览器兼容性问题。工具会自动使用备用方案生成.doc格式，Word可以正常打开。

### Q2: 可以批量转换吗？

**A**: 当前版本支持单次转换。如需批量，可以：
- 多次使用工具
- 或将多个Markdown内容合并后转换

### Q3: 转换后的格式不理想？

**A**: 
- TXT转换会去除所有格式，这是正常的
- Word转换会尽量保留格式，但复杂格式可能有限制
- 可以先用HTML预览效果

### Q4: 手机可以使用吗？

**A**: 可以！工具采用响应式设计，手机浏览器可以正常使用。

### Q5: 需要安装什么吗？

**A**: 不需要！这是纯前端工具，只需浏览器即可。
